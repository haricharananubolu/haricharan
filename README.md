# Grade Calculator

## Project Description

The Grade Calculator is a Python-based application developed to calculate and analyze student grades based on marks obtained in three subjects. The program allows the user to enter information for multiple students, validates the entered marks, calculates average marks, assigns grades, provides performance comments, and generates class statistics. The project demonstrates the use of fundamental Python concepts and helps improve problem-solving skills.

---

## Objectives

The main objectives of this project are:

* To understand the use of functions in Python.
* To practice conditional statements using if-elif-else.
* To learn how to use loops effectively.
* To store and process data using lists.
* To calculate average marks and class statistics.
* To understand searching techniques and file handling.
* To develop a simple and useful Python application.

---

## Features

* Accepts information for multiple students.
* Validates marks entered by the user.
* Calculates average marks for each student.
* Assigns grades based on average marks.
* Provides comments according to performance.
* Stores student information in lists.
* Displays results in a formatted report.
* Calculates class average.
* Finds highest and lowest averages.
* Searches for a specific student.
* Saves results to a file.
* Uses color coding for grades.

## Algorithm

1. Start the program.
2. Ask the user for the number of students.
3. Validate that the entered number is positive.
4. Collect names and marks for three subjects.
5. Validate that marks are between 0 and 100.
6. Calculate average marks for each student.
7. Determine grades and performance comments.
8. Store the results in a list.
9. Calculate class statistics.
10. Display all student results.
11. Search for a specific student.
12. Save the results to a text file.
13. End the program.
